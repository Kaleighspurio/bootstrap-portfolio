Booze and Brews
Project: Booze and Brews

This web application utilizes a few free APIs (OpenBrewerDB, CocktailDB, static GoogleMapsAPI) to allow the user to search for cocktail recipes by an ingredient or to find breweries by location. Styled with CSS and Bulma, and utilizing jQuery, this application was an early group project.

Task Scheduler
Daily Task Scheduler

This is a small project that utilizes local storage and Moment.js. The user can type in events for each hour and it save them to local storage once the Save button is clicked. As time goes by, various style changes occur to demonstrate when events are past, currently happening, or upcoming. Once the user no longer needs the events stored, they may clear the schedule, which clears local storage.

View the Github repository

Weather Dashboard
Weather Dashboard

This application utilizes the OpenWeatherAPI to obtain information about the current and 5 day forecast for the location that a user searches for. Upon the user entering a town and selecting a state, and the current weather and 5 day forecast elements will be dynamically created and appended to the webpage to display the weather information for the location searched. The user's last search is saved in local storage so that on revisiting or reloading the page, the same request is made of the API and the current and 5 day weather forecast are displayed.
